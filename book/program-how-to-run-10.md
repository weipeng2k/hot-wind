# 通过汇编语言了解程序的实际构成

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;汇编语言是由指令和操作数组成的语言，它面向硬件，编写汇编代码能够感受到直接和硬件对话的感觉，毕竟你操作的就是寄存器和内存。虽然语言显得很枯燥，但是通过一些模式，还是能够演绎出许多特性，朴实的表面下是强大的能力。
