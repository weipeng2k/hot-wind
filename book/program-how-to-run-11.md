# 硬件控制方法

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;端口是计算机对外部设备的区分定义，通过`IN`和`OUT`指令能够让计算机访问外部设备，当然有了操作系统的帮助，应用程序不会同硬件直接打交道，直接与操作系统的**API**对接即可。

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;外部设备请求**CPU**通过中断请求来提醒**CPU**进行介入。
